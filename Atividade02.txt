1. Criar um formulário de login com Ionic e JavaScript.

a) Abra o aplicativo VSCode.

b) Crie uma pasta Atividade02 acionando a opção New Folder sobre a pasta ADS023.

c) Crie um arquivo index.html acionando o menu New File com o botão direito sobre a pasta Atividade02.

d) Digite a estrutura básica de uma página HTML5.

e) Importe o CDN do Framework Ionic antes da tag </head>.

<script type="module" src="https://cdn.jsdelivr.net/npm/@ionic/core@4.7.4/dist/ionic/ionic.esm.js"></script>
<script nomodule src="https://cdn.jsdelivr.net/npm/@ionic/core@4.7.4/dist/ionic/ionic.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@ionic/core@4.7.4/css/ionic.bundle.css"/>

f) Crie um elemento ion-app dentro do corpo da página. 

g) Dentro do elemento ion-app crie um elemento um elemento ion-content.

h) Dentro do elemento ion-content crie um elemento ion-grid contendo um ion-row com um ion-col.

<ion-grid>
 <ion-row>
   <ion-col size-md="6" offset-md="3"></ion-col>
 </ion-row>
</ion-grid>

i) Dentro do elemento <ion-col> crie um elemento <ion-card> contendo um elemento <ion-card-header> e um elemento <ion-card-content>.
 
j) Dentro do elemento <ion-card-header> crie um elemento <ion-card-title> contendo o texto Aluno Online.

k) Dentro do elemento <ion-card-content> crie dois elementos <ion-item> e uma div.

<ion-item></ion-item>
<ion-item></ion-item>
<ion-item></ion-item>
<div margin-vertical text-center></div>

l) Dentro do primeiro <ion-item> crie um elemento <ion-label> e um elemento <ion-input> para o campo Matrícula.

<ion-label position="floating">Matrícula</ion-label>
<ion-input type="number"></ion-input>

m) Dentro do segundo <ion-item> crie um elemento <ion-label> e um elemento <ion-input> para o campo Senha.

<ion-label position="floating">Senha</ion-label>
<ion-input type="password"></ion-input>

n) Dentro da div adicione um elemento ion-button.

<ion-button color="danger">Entrar</ion-button>

o) Abra o arquivo index.hml no navegador Chrome. Ative as ferramentas do desenvolvedor pressionando a tecla F12 ou CTRL+SHIFT+I.

q) Pressione as teclas Ctrl+Shift+M para ativar o modo de visualização em dispositivos móveis. Escolha um tamanho para visualizar a página.
